# 须知

1. 纯纯看番是为了学习 Jitpack compose 和音视频相关技术进行开发的一个项目，官方不提供打包和下载，其源代码仅供交流学习。因其他人私自打包发行后造成的一切后果本方概不负责。
2. 纯纯看番打包后不提供任何视频内容，需要用户自己手动添加。用户自行导入的内容和本软件无关。
3. 纯纯看番源码完全免费，在 Github 开源。用户可自行下载打包。如果你是收费购买的本软件，则本方概不负责。
