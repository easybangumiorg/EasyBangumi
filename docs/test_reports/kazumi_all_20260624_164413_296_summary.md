## 后续排查点

- `ParserException`：本轮 21 个 block 源在请求失败分支出现 `Java constructor ... arguments "string" not found`。这不是站点搜索为空，而是异常构造方式和 Rhino/Java 构造器匹配仍有问题；本轮仅记录，不修改源。
- `RenderHelperImpl`：`dmghg/fcdm/fqdm/gugu3/skr/xfdmneo` 搜索和详情都正常，播放 URI 为空。优先用真实 WebView 抓取播放器跳转、iframe、wrapper query 和嗅探超时，再决定是工具问题还是源规则问题。
- 搜索为空源：需要逐个核对站点真实搜索页是否有 `孤独摇滚`，再区分 XPath 漂移、搜索 URL/编码变化、站点无结果。
- `kazumi-anfuns.js`：本轮不是空结果，而是 `unexpected end of stream`，建议单独复测以判断站点稳定性。
